jdk1.8安装步骤

  1. 去Orcale官网在页面中找到Java
    2.在页面中找到Oracle JDK

  2. 在页面中找到Java 8的对应版本下载

  3. 无脑点击下载的exe，不建议安装在C盘

  4. 环境配置

    1. 进入环境变量：我的电脑右键属性 -> 高级系统设置 -> 环境变量

    2. 配置JAVA_HOME：点击新建 变量：JAVA_HOME,值：JDK8安装目录

      解释:因为我安装了两个版本的jdk所以再间接的包一层版本HOME

    3. 配置CLASSPATH

    4. 配置Path

  5. 检验配置是否成功
    打开cmd输入

  • java
  • ``````javac```
  • java -version这显示我的jdk版本为17可能因为我用的是17
